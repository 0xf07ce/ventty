# KGM - MDIR Clone for Linux

1990년대 한국에서 널리 사용되던 MS-DOS 셸 프로그램 MDIR의 외형과 사용성을 현대 Linux/Unix 환경에서 재현한 터미널 기반 파일 관리자.

## 빌드

### 요구사항

- C++20 지원 컴파일러 (GCC 10+, Clang 12+)
- CMake 3.16+

### 빌드 방법

```bash
# 설정
cmake -B cmake-build-debug -S .

# 빌드
cmake --build cmake-build-debug

# 실행
./cmake-build-debug/src/kgm
```

## 현재 기능 (v0.3-dev)

### 파일 탐색
- 디렉토리 목록 표시 (MDIR 스타일)
- 파일 타입별 색상 구분 (디렉토리, 실행파일, 압축파일 등)
- 키보드 네비게이션 (↑↓, PgUp/PgDn, Home/End)
- 디렉토리 진입 (Enter)
- 상위 디렉토리 이동 (Backspace)
- 숨김 파일 토글 (Ctrl+H)

### 파일 선택
- 단일 파일 선택 (Space, Insert)
- 전체 선택 (Ctrl+A)
- 전체 해제 (Ctrl+D)
- 선택 반전 (*)

### 파일 조작
- 파일/디렉토리 삭제 (F8)
- 디렉토리 생성 (F7)
- 파일 실행 (Enter)

### 메뉴 시스템
- 메뉴 바 (F12)
- 드롭다운 메뉴

### 기타
- True Color (24-bit) 지원
- UTF-8 한글 파일명 지원
- 터미널 크기 자동 조정

## 키 바인딩

| 키 | 동작 |
|----|------|
| ↑/↓ | 파일 선택 이동 |
| PgUp/PgDn | 페이지 이동 |
| Home/End | 처음/끝으로 이동 |
| Enter | 디렉토리 진입 / 파일 실행 |
| Backspace | 상위 디렉토리 |
| Space/Insert | 파일 태그 토글 |
| Ctrl+A | 전체 선택 |
| Ctrl+D | 전체 해제 |
| * | 선택 반전 |
| Ctrl+H | 숨김 파일 토글 |
| F5 | 새로고침 |
| F7 | 디렉토리 생성 |
| F8 | 삭제 |
| F12 | 메뉴 |
| q / Alt+X | 종료 |
