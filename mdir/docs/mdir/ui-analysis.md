# MDIR UI 분석 (스크린샷 기반)

## 1. 메인 화면 분석

### 1.1 기능키 바 (행 0)

```
F2OCPlay │ 3EDIT │ 4UEDIT │ 5Virus │ 6SAN │ 7pctools │ 8HELP │ 9Defrag
```

- 숫자: 빨간색 강조
- 라벨: 검정 배경에 시안색 텍스트
- 구분: 세로선(│)으로 분리

### 1.2 경로/볼륨 바 (행 1)

```
Path C:\*.*                                           Volume [C_DRIVE]
```

- "Path": 시안색
- 경로: 흰색
- "Volume": 시안색
- 볼륨명: 흰색, 대괄호로 감싸짐

### 1.3 파일 목록 영역 (행 2-20)

```
테두리: 시안색 박스 문자 (─│┌┐└┘)
배경: 검정
```

### 1.4 파일 항목 색상

| 항목 유형 | 색상 (Foreground) | 예시 |
|----------|------------------|------|
| .. (상위 디렉토리) | 흰색 | `..          [ Up-Dir ]` |
| 서브디렉토리 | 노란색 | `GAME        [ SubDir ]` |
| EXE 파일 | 초록색 | `PROGRAM  EXE` |
| COM 파일 | 밝은 파란색 | `COMMAND  COM` |
| BAT 파일 | 노란색 | `AUTOEXEC BAT` |
| 일반 파일 | 흰색/회색 | `CONFIG   SYS` |

### 1.5 선택 항목

- 배경: 빨간색
- 텍스트: 원래 색상 유지

### 1.6 드라이브 목록 (행 21-22)

```
[-A-] Remove
[-C-] HDD C_DRIVE
[-Y-] HDD ESSENTIALS
[-Z-] HDD DOSBOX
```

- 드라이브 문자: 노란색, 대괄호 포함
- 설명: 흰색/회색

### 1.7 상태 바 (행 23)

```
 6 File    6 Dir         39,350 Byte    262,144,000(48%)byte free (511M) Q
```

- 배경: 시안색
- 텍스트: 검정색
- 숫자: 강조

### 1.8 하단 정보 바 (행 24)

```
Directory│26-01-28│22:07│....│26-03-25│WED│10:19:40pm│F10=MCD│F11=QCD│F12=Menu
```

- 배경: 시안색/빨간색 혼합
- 기능키: 노란색 강조

## 2. 메뉴 화면 분석

### 2.1 메뉴 바

```
     Mdir    Run    File  Direct Archive    Option
```

- 배경: 시안색
- 선택된 메뉴: 검정 배경
- 단축키 문자: 빨간색 (예: "M"dir)

### 2.2 드롭다운 메뉴

```
┌────────────────────────────┐
│ About                      │
│ Help                    F1 │
│ View environment           │
├────────────────────────────┤
│ Parking              Ctrl-P│
│ Volume label         Ctrl-V│
│ Format               Ctrl-F│
│ diskCopy             Ctrl-C│
│ Screen saver         Ctrl-S│
├────────────────────────────┤
│ Dos command              / │
│ Dos command(swap)        ? │
├────────────────────────────┤
│ Quit to dos          Alt-X │
└────────────────────────────┘
```

- 테두리: 시안색
- 배경: 시안색
- 텍스트: 검정색
- 단축키: 우측 정렬, 검정색
- 구분선: 수평선(─)

## 3. MCD 다이얼로그 분석

```
┌─ MdirIII Change Directory ───────────────────────────────────────┐
│                                                                  │
│ \                                                                │
│ ├─ MDIR ─── BABO ─── LOVE ─── M...                               │
│ ├─ GOOD ─── UTIL ─── OUCH ─── M...                               │
│ └─ HELP ─── GAME                                                 │
│                                                                  │
│ C:\MDIR                                              [MDIR     ] │
├──────────────────────────────────────────────────────────────────┤
│ F1=Help │ F2=Rescan │ F3=Drive │ F4=Move                         │
└──────────────────────────────────────────────────────────────────┘
```

- 타이틀: 상단 테두리에 삽입
- 트리: 박스 문자(├─└)로 구성
- 현재 디렉토리: 노란색 강조
- 선택된 디렉토리: 분홍/마젠타 배경

## 4. 박스 문자 (Code Page 437)

```

├ (195) ─ (196) ┤ (180)
│ (179)         │ (179)
└ (192) ─ (196) ┘ (217)
```

## 5. 색상 팔레트 (True Color 매핑)

| 인덱스 | DOS 색상 | RGB 값 | 용도 |
|--------|----------|--------|------|
| 0 | Black | #000000 | 배경 |
| 1 | Blue | #0000AA | - |
| 2 | Green | #00AA00 | EXE 파일 |
| 3 | Cyan | #00AAAA | 테두리, 상태바 |
| 4 | Red | #AA0000 | 선택 배경 |
| 5 | Magenta | #AA00AA | 압축 파일 |
| 6 | Brown | #AA5500 | - |
| 7 | Light Gray | #AAAAAA | 일반 파일 |
| 8 | Dark Gray | #555555 | 비활성 |
| 9 | Light Blue | #5555FF | COM 파일 |
| 10 | Light Green | #55FF55 | - |
| 11 | Light Cyan | #55FFFF | - |
| 12 | Light Red | #FF5555 | - |
| 13 | Light Magenta | #FF55FF | - |
| 14 | Yellow | #FFFF55 | 디렉토리, BAT |
| 15 | White | #FFFFFF | 선택 항목 |

### 5.2 스크린샷 실제 색상 (측정값)

| 요소 | 실제 RGB | 설명 |

| 빨간 선택 | #AA0000 | 진한 빨강 |
| 흰색 텍스트 | #FFFFFF | 순수 흰색 |
