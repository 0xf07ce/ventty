## 파일 선택 시스템

### 선택 상태

```cpp
class FilePanel
{
    // 기존 멤버
    std::set<int> _selectedIndices;  // 다중 선택된 인덱스

public:
    void toggleSelection(int index);
    void selectAll();
    void deselectAll();
    void invertSelection();

    std::vector<kgm::fs::FileInfo const *> selectedFiles() const;
    bool hasSelection() const { return !_selectedIndices.empty(); }
};
```

### 선택 표시

- 선택된 파일: 노란색 배경 또는 `*` 마커
- 현재 커서: 빨간색 배경
- 선택 + 커서: 밝은 빨간색 배경

### 키 바인딩

| 키 | 동작 |
|----|------|
| Space | 현재 파일 선택/해제 후 아래로 이동 |
| Insert | 현재 파일 선택/해제 후 아래로 이동 |
| Ctrl-A | 전체 선택 |
| Ctrl-D | 전체 해제 |
| * | 선택 반전 |

### 복사 로직

```cpp
void Application::copyFiles()
{
    auto files = getFilesToCopy();  // 선택된 파일 또는 현재 파일
    if (files.empty()) return;

    std::string dest = showInputDialog("Copy",
        "Copy " + std::to_string(files.size()) + " file(s) to:",
        _filePanel->fileSystem().currentPath().string());

    if (dest.empty()) return;

    ProgressDialog progress;
    progress.setTitle("Copying");

    for (size_t i = 0; i < files.size(); ++i)
    {
        progress.setFile(files[i]->name);
        progress.setProgress(i * 100 / files.size());

        // 복사 수행
        std::error_code ec;
        std::filesystem::copy(files[i]->path, dest / files[i]->name, ec);

        if (ec)
        {
            // 에러 처리
        }
    }

    _filePanel->refresh();
}
```

## ProgressDialog 위젯

```cpp
class ProgressDialog : public Widget
{
public:
    void setTitle(std::string title);
    void setFile(std::string filename);
    void setProgress(int percent);  // 0-100
    void setMessage(std::string msg);

    bool isCancelled() const { return _cancelled; }

    void draw(Screen & screen) override;
    bool handleKey(KeyEvent const & event) override;

private:
    std::string _title;
    std::string _filename;
    std::string _message;
    int _progress = 0;
    bool _cancelled = false;
};
```

### 진행률 바 그리기

```cpp
void ProgressDialog::drawProgressBar(Screen & screen, int x, int y, int width, int percent)
{
    int filled = width * percent / 100;

    screen.setStyle({Colors::GREEN, Colors::BLACK});
    for (int i = 0; i < filled; ++i)
    {
        screen.putChar(U'█');
    }

    screen.setStyle({Colors::DARK_GRAY, Colors::BLACK});
    for (int i = filled; i < width; ++i)
    {
        screen.putChar(U'░');
    }
}
```
