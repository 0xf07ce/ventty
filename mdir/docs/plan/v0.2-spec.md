## 파일 조작

### 삭제 (F8)

1. 선택된 파일/디렉토리 확인
2. 확인 다이얼로그 표시 ("Delete {filename}?")
3. 삭제 수행
4. 파일 목록 새로고침

```cpp
void Application::deleteSelected()
{
    auto const * file = _filePanel->selectedFile();
    if (!file || file->isParentDir())
    {
        return;
    }

    Dialog confirm;
    confirm.setTitle("Delete");
    confirm.setMessage("Delete '" + file->name + "'?");
    confirm.addButton("Yes", DialogResult::Yes);
    confirm.addButton("No", DialogResult::No);

    if (confirm.show() == DialogResult::Yes)
    {
        std::error_code ec;
        std::filesystem::remove_all(file->path, ec);
        _filePanel->refresh();
    }
}
```

### 디렉토리 생성 (F7)

1. 입력 다이얼로그 표시 ("Directory name:")
2. 디렉토리 생성
3. 파일 목록 새로고침

```cpp
void Application::makeDirectory()
{
    InputDialog input;
    input.setTitle("Make Directory");
    input.setPrompt("Directory name:");

    if (input.show() == DialogResult::Ok)
    {
        std::string name = input.value();
        if (!name.empty())
        {
            auto path = _filePanel->fileSystem().currentPath() / name;
            std::error_code ec;
            std::filesystem::create_directory(path, ec);
            _filePanel->refresh();
        }
    }
}
```
